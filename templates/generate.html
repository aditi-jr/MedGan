How to Use?

Select the Model

Choose the GAN model you want to use from the "Model" dropdown menu.

Select the Tumor Type

Pick the tumor type (Glioma, Pituitary, or Meningioma) from the "Tumor Type" dropdown.

Set the Number of Images

Enter the number of images to generate in the "Number of Images" field (1–100).

Generate Images

Click the "Generate Images" button to create your selected images.

Generate Brain Tumor Images

Select a model, tumor type, and the number of images to generate.
